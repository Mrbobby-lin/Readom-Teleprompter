# 🎤 语音提词器 - 开发文档

## 项目概述

Windows 桌面提词器软件。用户导入演讲稿（txt / Word / 粘贴），对着麦克风朗读，软件自动识别语音并高亮对应文字，读到哪里 Highlight 到哪里。

## 技术栈

| 模块 | 技术 |
|------|------|
| UI 框架 | PySide6 (Qt6) |
| 离线语音 | faster-whisper（原 Vosk → 已替换） |
| 在线语音 | 百度语音 API |
| 音频采集 | PyAudio |
| Word 解析 | python-docx |
| 打包工具 | PyInstaller |

## 架构设计

### 数据流

```
用户点击"开始"
    │
    ▼
Recorder (PyAudio) → 麦克风音频流
    │
    ▼
Speech Engine (Vosk / 百度 API) → 识别文字
    │
    ▼
检查语音命令 ←──→ Matcher (3级匹配) → 找到对应位置
    │
    ▼
MainWindow → 更新高亮 + 滚动 + 进度条
```

### 文本匹配策略（三级递进）

1. **段落级** - 与段落开头比较相似度，确定在哪个段落
2. **句子级** - 在段落内逐句比较字符重叠率 + Jaccard 相似度
3. **字符级** - 在句子内追踪已识别字符比例

### 状态机

```
IDLE ──[点击 开始]──→ LISTENING
LISTENING ──[点击 暂停 / 语音"暂停"]──→ PAUSED
PAUSED ──[点击 继续 / 语音"继续"]──→ LISTENING
LISTENING ──[点击 停止 / 语音"停止"]──→ IDLE
```

### 语音命令

| 命令 | 触发词 |
|------|--------|
| 暂停 | "暂停", "停一下", "停", "歇一下" |
| 继续 | "继续", "开始", "恢复" |
| 后退 | "后退", "上一句", "回退", "往前" |
| 前进 | "前进", "下一句", "往后" |
| 停止 | "停止", "结束", "关闭" |

### 布局

- **全屏模式**：文字区域居中显示，当前句子在屏幕垂直居中位置
- **顶部栏**：鼠标移到顶部时显示（导入、设置、退出全屏）
- **底部栏**：控制按钮（后退、暂停/继续、前进、停止）+ 进度条

## 打包说明

```bash
# 安装依赖
pip install -r requirements.txt

# 下载 Vosk 中文模型（可选）
# 从 https://alphacephei.com/vosk/models 下载 vosk-model-cn-0.22
# 解压到 resources/ 目录

# 打包
python package.py
# 产物在 dist/语音提词器.exe

# 清理打包产物
python package.py --clean
```
